# Bun + Nestjs

To install dependencies:

```bash
bun install
```

To run tests:

```bash
bun test
```

To run development:

```bash
bun run dev
```

To run debug:

```bash
bun run debug
```

To run production:

```bash
bun run production
```

This project was created using `bun init` in bun v1.0.3. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
